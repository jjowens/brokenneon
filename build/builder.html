NeonBuilder

Broken Neon Lights

Neon Builder allows you to dynmically create text shadows inside an array. Pass it into the API and it will store the array in memory. It will build up a long string from each element in the array. That long string will be applied as a textShadow style

As the array is held in memory, you can add, insert or remove text shadows. It will automatically update the textshadows.

Please note you need to create an instance of NeonBuilder.
