BrokenNeon API

List of API functions

FlickerBroken - One or two text characters will flicker on and off to create a broken lights effect

NeonBuilder - Create an array of textShadows and apply it to a text. You can add, insert or remove textShadows and it will automatically update the neon lights effect

RandomFlicker - Each character will light up one by one. It'll start with the first character before stopping on the last one. You can reset the effect every time it ends.

ToggleGradual - Each character will light up one by one. It'll start with the first character before stopping on the last one. You can reset the effect every time it ends.

ToggleLights - Each character will light up one by one. It'll start with the first character before stopping on the last one. You can reset the effect every time it ends.

Complex Example 1 - Uses a combination of API calls. This turns off all lights, then it will gradually light up each up each character. Finally, one of the lights will flicker as it's broken.
